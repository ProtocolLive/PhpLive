PhpLiveDbBackup::__construct

Creates a PhpLiveDbBackup instance.

 

Sintaxe

public PhpLiveDbBackup::__construct ( [ object &$PhpLivePdo = null ] );

 

Parameters

&$PhpLivePdo

The PhpLivePdo object reference. If the object are not informed in construct, its must be informed in one of the functions of the class.

 

Return Values

Returns a PhpLiveDbBackup object on success.

 

Note

The PhpLiveForms system use the session_name() to identify the forms as "Site" parameter.

 

Dependencies

- PHP 7;

- PhpLivePdo library;

 

Examples

$backup = new PhpLiveDbBackup($PDO);
