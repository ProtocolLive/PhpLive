PDO - Protocol PHP Live

SqlConnect

Creates a connection to the database.

Description

SqlConnect(array $Options) : object

Parameters

Drive (Optional)
Select the database (List)
The default value is "mysql"

Ip
The data base ip or dns

Db
The default database schema

Charset (Optional)
The default characters set
The default value is "utf8"

User
User name to log in database

Pwd
User password to log in database

TimeOut (Optional)
The maximun time to any execution, in seconds
The default value is "5"

Prefix (Optional)
The prefix of the tables

Returned value

Returns a PDO object on success.

Errors

SqlConnect() throws a PDOException if the attempt to connect to the requested database fails.
If ini configuration "display_errors" are set to false, will be displayed a friendly error description in Brazil portuguese.

Explanations

The parameters are send in array mode, because it is easier. You can send the parameters in any order

Examples

$conn = SqlConnect([
   "Ip" => "localhost",
   "User" => "root",
   "Pwd" => "password",
   "Db" => "mysite"
]);
