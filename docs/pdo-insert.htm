PhpLivePdo::Insert

Execute a insert query. This function its good to avoid differences when specifying tokens.

 

Sintaxe

public PhpLivePdo::Insert ( array $Options [, array $Options2 = [] ] ) : int;

 

Parameters

$Options:

  Table (string) - The table afected;

  Fields (array) - The fields to be added;

Sintaxe of Fields option:

[TokenName, TokenData, TokenType]

TokenName (int or string) - A number or a string starting with ":";

TokenData (int or string) - The value to be replaced;

TokenType (int) - The constants of PhpLivePdo (PdoInt, PdoStr, PdoNull, PdoBool, PdoSql) or the constants of PDO;

 

$Options2 are the $Options in the PhpLivePdo::Run function

 

Return Values

Returns the last auto_increment value.

 

Examples

$PDO->Insert([
   "Table" => "users",
   "Fields" => [
      ["email", "a@a.com", PdoStr],
      ["pwd", "123", PdoStr],
      ["name", "adm", PdoStr],
      ["level", 1, PdoInt]
   ]
]);
// insert into users(email,pwd,name,level) values('a@a.com','123','adm',1)
