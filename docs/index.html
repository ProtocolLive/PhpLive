PhpLive

PhpLive is a series of free libraries in PHP to anyone use in they own projects:

PhpLiveCpf - A function to check the verificator code of brazillian register CPF;

PhpLiveDbBackup - A class to backup the database;

PhpLiveForms - A class to build simples forms;

PhpLivePdo - A class to work with database;

PhpLivePerms - A class to create a system of access permissions;

PhpLivePhpUpdate - A function to check the last available version of PHP;
