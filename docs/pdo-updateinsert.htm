PhpLivePdo::UpdateInsert

Run a update query or a insert query in case of the where clause is not satisfied

 

Sintaxe

public PhpLivePdo::UpdateInsert ( array $Options [, array $Options2 = [] ] ) : int;

 

Parameters

List of $Options:

  Table - The table afected;

  Fields - The fields to be changed;

  Where - The conditions;

$Options2 are the $Options in the SQL function

 

Return Values

Returns the number of rows affected.

 

Atention

- You have to specify the parameters as if you were going to use the function SqlInsert, in case it have to run this function;

 

Limitations

- The clause where only use "=" comparator;

- The clause where only use one condition;

 

Examples

// Original data: email=a@a.com,pwd=123,name='adm'
$PDO->SqlUpdate([
   "Table" => "users",
   "Fields" => [
      ["email", "a@a.com", PdoStr],
      ["pwd", "321", PdoStr],
      ["name", "adm", PdoStr]
   ],
   "Where" => ["user_id", $_GET["id"], PdoInt]
]);
// Run: update users pwd='321' where user_id=1
