PhpLivePdo::SqlInsert

Execute a insert query. This function its good to avoid differences when specifying tokens.

 

Sintaxe

public PhpLivePdo::SqlInsert ( array $Options [, array $Options2]);

 

Parameters

List of $Options:

  Table - The table afected;

  Fields - The fields to be added;

$Options2 are the $Options in the SQL function

 

Return Values

Returns the last auto_increment value.

 

Examples

$PDO->SqlInsert([
   "Table" => "users",
   "Fields" => [
      ["email", $_POST["email"], PdoStr],
      ["pwd", $_POST["pwd"], PdoStr],
      ["name", $_POST["name"], PdoStr],
      ["level", $_POST["level"], PdoInt]
   ]
]);
// insert into users(email,pwd,name,level) values('a@a.com','123','adm',1)
