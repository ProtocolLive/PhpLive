PhpLivePdo::SqlUpdate

Execute a update query. This function its good to avoid differences when specifying tokens.

 

Sintaxe

public PhpLivePdo::SqlUpdate ( array $Options [, array $Options2 = [] ] ) : int;

 

Parameters

List of $Options:

  Table - The table afected;

  Fields - The fields to be changed;

  Where - The conditions;

$Options2 are the $Options in the SQL function

 

Return Values

Returns the number of rows affected.

 

Limitations

- The clause where only use "=" comparator;

- The clause where only use one condition;

 

Examples

$PDO->SqlUpdate([
   "Table" => "users",
   "Fields" => [
      ["email", "a@a.com"], PdoStr],
      ["pwd", "123", PdoStr],
      ["name", "adm", PdoStr]
   ],
   "Where" => ["user_id", 1, PdoInt]
]);
// update users set email='a@a.com',pwd='123',name='adm' where user_id=1
