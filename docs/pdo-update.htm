PhpLivePdo::Update

Run a update query afecting only the differents fields. This resource its good to safe space in log.

 

Sintaxe

public PhpLivePdo::Update ( array $Options [, array $Options2 = [] ] ) : int;

 

Parameters

$Options:

  Table (string) - The table afected;

  Fields (array) - The fields to be changed;

  Where (array) - The conditions;

Sintaxe of Fields and Where options:

[TokenName, TokenData, TokenType]

TokenName (int or string) - A number or a string starting with ":";

TokenData (int or string) - The value to be replaced;

TokenType (int) - The constants of PhpLivePdo (PdoInt, PdoStr, PdoNull, PdoBool, PdoSql) or the constants of PDO;

 

$Options2 are the $Options in the PhpLivePdo::Run function

 

Return Values

Returns the number of rows affected.

 

Limitations

- The clause where only use "=" comparator;

- The clause where only use one condition;

 

Examples

// Original data: email=a@a.com,pwd=123,name='adm'
$PDO->Update([
   "Table" => "users",
   "Fields" => [
      ["email", "a@a.com", PdoStr],
      ["pwd", "321", PdoStr],
      ["name", "adm", PdoStr]
   ],
   "Where" => ["user_id", 1, PdoInt]
]);
// Run: update users pwd='321' where user_id=1
