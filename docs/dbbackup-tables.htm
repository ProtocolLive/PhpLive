PhpLiveDbBackup::Tables

Make backup of the structure of the tables.

 

Sintaxe

public PhpLiveDbBackup::Tables ( [ array $Options = [] ] ) : string;

 

Parameters

$Options:

Folder (string)(Optional - Default: /sql/) - A directory to put the files;

Progress (bool)(Optional - Default: true) - Dump the progression of the backup;

Translate (array)(Optional) - Text to translate the progress dump;

Parameters of Translate:

Tables (string) - Text of the tables count;

FK (string) - Text of the foreign keys;

 

Return Values

Returns the url to download the backup.

 

Examples

$url = $backup->Tables();
printf('<a href="%s">Download</a>', $url);
